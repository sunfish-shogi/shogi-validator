# 目的

Web ブラウザで利用可能な将棋の棋譜データや局面データの検査ツールを提供する。

# インターフェース

単一のWebページ内で以下のコンポーネントを配置する。

- データ入力欄（テキストエリア）
- ファイル選択ボタン
- 検査実行ボタン
- 検査結果
- 各種フォーマットへの変換結果（コピーボタン付き）

# 機能

## P0

以下のフォーマットをサポートする。

- KIF
- KI2
- CSA
- JKF
- USI
- SFEN
- USEN

どのフォーマットかを自動で判定し、その結果を表示する。

## P1

tsshogi でパースできない場合はエラーの内容を確認できる。

## P2

他のフォーマットへ変換した結果を出力する。

## P3

ファイル読み込みに対応する。
フィアルを読み込んだ場合は文字コードを自動判別し、その結果を表示する。

## P4

KIF と KI2 の場合は柿木将棋の出力と違う点（スペースの入れ方や空行の入れ方など）を評価し、一致度 (%) と相違点を説明する。
